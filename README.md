# ToolX-EdgeSync-DCA-6101

## Overview

ToolX-EdgeSync-DCA-6101 is  scalable software infrastructure for managing over-the-air (OTA) firmware updates across industrial edge devices running embedded Linux. The system will streamline update deployment, ensure end-to-end security, and support future growth with modular architecture and cloud-native components.

[Try it Out!](https://toolxedgesync-g0ekfec6d3ayeubw.eastus-01.azurewebsites.net/)

## Getting Started

### Prerequisites

- Python 3.8+
- Node.js 16+

### Installation

**1. Install Python dependencies:**
```bash
pip install -r requirements.txt
```

**2. Start the frontend:**
```bash
npm run dev
```

**3. Start the backend:**
```bash
python -m uvicorn main:app --reload
```
